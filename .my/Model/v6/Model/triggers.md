# External Triggers

Discrete events that change transition probabilities. ⚡ markers in state descriptions point to the most common trigger types per state.

## ⚡ marker rule

A state carries ⚡ if and only if at least one named external event materially changes the probability of entering or leaving that state. States that change only through internal cognition or ambient social context do not carry ⚡.

## Common triggers

| Trigger | Typical effect |
| --- | --- |
| Model release | Forward push (S2D → S3E) or regression (S5/S7 → S3E) |
| Workplace mandate | Compresses S0–S2D; can force S5 without S3E |
| Media narrative | Amplifies whichever state matches the narrative's framing |
| AI failure (visible) | Regression from any state; reinforces S2D |
| Regulatory change | Freezes adoption or forces re-evaluation |
| Personal underperformance | Re-triggers S3E from any S5–S7 state |
| Peer adoption shift | Triggers S6S; can compress S2D |
| Cost or access change | Holds people in upstream states for structural rather than psychological reasons |

## Disambiguation rule

Dated event → external trigger layer (this file). Undated ambient condition → see [social-context.md](social-context.md).
